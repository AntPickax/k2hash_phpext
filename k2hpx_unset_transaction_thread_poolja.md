# k2hpx_unset_transaction_thread_pool
トランザクション処理用のスレッドを利用しない

## 説明

```
bool k2hpx_unset_transaction_thread_pool ( void )
```

トランザクション処理用のスレッドを利用しません。 

## パラメータ
この関数にはパラメータはありません。

## 戻り値
成功した場合に true を、失敗した場合に false を返します。
