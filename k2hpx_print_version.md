# k2hpx_print_version
Prints the library version

## Description

```
bool k2hpx_print_version ([ mixed $output ] )
```

Prints the library version. 

## Parameters
- output  
Specifies a file stream of data to be written.

## Return Values
Returns true on success or false on failure.
