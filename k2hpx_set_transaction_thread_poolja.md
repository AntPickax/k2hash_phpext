# k2hpx_set_transaction_thread_pool
トランザクション処理用のスレッド数を設定する

## 説明

```
bool k2hpx_set_transaction_thread_pool ( int $count )
```

トランザクション処理用のスレッド数を設定します。 

## パラメータ
- count  
トランザクション処理用のスレッド数

## 戻り値
成功した場合に true を、失敗した場合に false を返します。
